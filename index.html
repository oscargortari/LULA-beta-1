Controles

Clic sostenido: arrastrar nodo

Doble clic: editar / seleccionar nodo

⌘ + arrastrar: selección múltiple

Shift + L: crear conexión

Shift + U: borrar conexión

Shift + N: crear nodo hijo

Shift + Z + mover arriba/abajo: ajustar profundidad Z

Botón 3D/2D: alternar vista espacial

Editor ON/OFF: mostrar u ocultar ayudas de edición

Cmd + S: guardar proyecto JSON
